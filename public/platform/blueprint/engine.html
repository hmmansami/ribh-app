المقارنة — كيف كل منصة تحل المشكلة
Klaviyo
  • نماذج التسجيل (Popups, Embeds, Flyouts)
  • أكثر من 350 تكامل (Integration)
  • تتبع الموقع عبر JavaScript
  • يعتمد على أن الزائر يعبي النموذج بنفسه
Attentive
  • تسجيل بنقرتين (Two-Tap) — براءة اختراع
  • Enhanced Tag — تتبع من السيرفر
  • Identity Graph — خريطة هوية العميل
  • AI Grow + عجلة الحظ (Spin-to-Win)
  • نافذة الخروج (Exit Intent)
رِبح — المدمج MERGED
  • Salla OAuth = بيانات كل العملاء فوراً بدون نماذج
  • نافذة خروج مع اشتراك واتساب — نقرة وحدة بس
  • Webhooks من سلة تلتقط كل حدث سلة لحظياً
  • أرقام الجوال موجودة أصلاً في سلة — ما تحتاج تجمعها
  • واتساب أبسط من SMS — ما يحتاج رمز تأكيد

Layer 2 — KNOW
الحقيقة الجوهرية — FUNDAMENTAL TRUTH
شخص واحد = سجل واحد. البيانات المتفرقة ضوضاء بلا قيمة.
المقارنة — كيف كل منصة تحل المشكلة
Klaviyo
  • منصة بيانات العملاء (CDP) — ملفات موحدة
  • أكثر من 350 مصدر بيانات مدمج
  • احتفاظ بالبيانات بلا حدود زمنية
  • حذف التكرار عبر مصادر متعددة (Deduplication)
Attentive
  • Identity Graph — الهاتف هو المعرف الأساسي
  • ربط الأجهزة المتعددة (Cross-Device Matching)
  • Rich Data Capture — يكمل بيانات الزوار المجهولين
  • ملف عميل واحد عبر كل القنوات
رِبح — المدمج MERGED
  • ملف عميل موحد من بيانات سلة (طلبات، منتجات، عناوين)
  • رقم الجوال هو المعرف الأساسي (صيغة 966+) — نفس فلسفة Attentive
  • تقييم RFM من اليوم الأول — بدون انتظار 180 يوم مثل Klaviyo
  • تاريخ المشتريات + السلات + الرسائل في سجل واحد
  • تخزين مزدوج: Firestore + ملفات JSON المحلية

Layer 3 — PREDICT
الحقيقة الجوهرية — FUNDAMENTAL TRUTH
البيانات بدون ذكاء = ضوضاء. القيمة الحقيقية هي معرفة إيش تسوي.
المقارنة — كيف كل منصة تحل المشكلة
Klaviyo
  • تحليلات تنبؤية بالتعلم الآلي (CLV, Churn, Next Order)
  • يحتاج أكثر من 500 عميل و180 يوم للتفعيل
  • تقسيم RFM — شرائح ديناميكية لحظية
  • Smart Send Time — توقيت إرسال ذكي
  • مقارنة مع 183 ألف متجر (Peer Benchmarking)
Attentive
  • ذكاء اصطناعي مدرب على 1.4 تريليون نقطة بيانات
  • AI Journeys — تخصيص تلقائي بالكامل
  • Audiences AI — شرائح ذكية
  • Send Time AI — أفضل وقت إرسال
  • Brand Voice AI — صوت العلامة التجارية
  • منع الإرسال التنبؤي (Predictive Suppression)
رِبح — المدمج MERGED
  • تقسيم RFM فوري من اليوم الأول (Champions, Loyal, At Risk)
  • توليد عروض بالذكاء الاصطناعي عبر Groq (مجاني) مع Gemini احتياطي
  • تحليلات تنبؤية: CLV، احتمالية المغادرة، الطلب التالي
  • إطار A/B Testing — اختبار كل شيء
  • حماية ذكية ضد الحظر (Anti-Ban) — تأخيرات بشرية
  • تعلم أفضل قناة لكل عميل (واتساب / إيميل / SMS)

Layer 4 — MESSAGE
الحقيقة الجوهرية — FUNDAMENTAL TRUTH
الذكاء بدون فعل = هدر. الرسالة هي اللي تجيب الإيراد.
المقارنة — كيف كل منصة تحل المشكلة
Klaviyo
  • أكثر من 80 قالب رحلة جاهز
  • محرر سحب وإفلات للإيميلات + 350 قالب
  • تفرعات شرطية + اختبار A/B
  • إيميل + SMS + Push + واتساب (تجريبي) + RCS (تجريبي)
  • K:AI Marketing Agent — وكيل تسويق ذكي
Attentive
  • AI Journeys — رحلات ذاتية التشغيل بالكامل
  • Magic Message — نصوص وصور بالذكاء الاصطناعي
  • اتصال مباشر مع شركات الاتصالات (SMS)
  • أولوية تلقائية بين الرحلات
  • محرك امتثال + Litigator Defender
رِبح — المدمج MERGED
  • بدل 350 قالب — الذكاء الاصطناعي يولد قوالب عربية لحظياً عبر Groq
  • بدل بناء رحلات يدوي — زر تشغيل/إيقاف (سلات متروكة، ترحيب، استرجاع)
  • واتساب أولاً (98% معدل فتح في السعودية) مع إيميل + SMS احتياطي
  • عروض مخصصة لكل عميل بالذكاء الاصطناعي
  • تسلسل متعدد الخطوات: واتساب ← إيميل ← SMS
  • محرك دورة الحياة ينسق كل شيء تلقائياً

المقارنة — كيف كل منصة تحل المشكلة
Klaviyo
  • إسناد إيراد متعدد نقاط اللمس (Multi-Touch Attribution)
  • Revenue Per Recipient — إيراد لكل مستلم
  • مقارنة مع 183 ألف متجر (Peer Benchmarking)
  • تقارير مخصصة + نتائج A/B
Attentive
  • تجارة محادثات (Concierge) — ردود ذكية ثنائية الاتجاه
  • Text-to-Buy — الشراء عبر رسالة نصية
  • المشتركين ينفقون 50% أكثر
  • إسناد: يوم مشاهدة / 5 أيام نقرة
رِبح — المدمج MERGED
  • إسناد قصصي: "أحمد — سلة 340 ر.س ← عرض 10% ← اشترى" — مش بس أرقام
  • محادثات واتساب ذكية ثنائية الاتجاه مثل Attentive Concierge
  • بيع إضافي بعد الشراء (Post-Purchase Upsell)
  • استرداد تصفح المنتجات (Browse Abandonment)
  • جمع تقييمات عبر واتساب
  • تتبع أحداث كامل للقمع — تأثير الشبكة عبر متاجر سلة
