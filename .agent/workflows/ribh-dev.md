// turbo-all

## Development Flow
1. Make code changes
2. Git add and commit changes
3. Git push to origin main
4. Deploy to Render (if API available)

## Testing
1. Run local server: npm start
2. Test endpoints with curl
3. Check logs for errors

## Common Commands
- `git status` - Check current state
- `git log -3` - See recent commits
- `curl https://ribh.click/api/stats` - Test live API
